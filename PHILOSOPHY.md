
## The Manifesto

We believe:

1. **Computation is physics.** Code that ignores thermal limits is fantasy.

2. **Flux over Functions.** Information flows like water. Block the flow, and everything dies.

3. **Feedback over Feedforward.** Neural networks are 40 years old. Coupled systems are eternal.

4. **Degradation as Signal.** When the system glitches, it's not failing—it's **communicating**.

5. **Legacy Hardware as Future.** An i5-2012 with intention beats a GPU farm with no soul.

6. **Hobbies over Careers.** The best code is built by people with nothing to lose and everything to prove.
